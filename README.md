# Connectivity Check

A simple JavaScript-based web application that checks and displays the user's internet connectivity status in real time. The application detects whether the device is online or offline using the browser's built-in `navigator.onLine` API and the `online`/`offline` events, instantly updating the user interface with the current network status.

This project was built using **HTML, CSS, and JavaScript** and version-controlled with **Git**. The source code is managed locally and uploaded to **GitHub** using Git, demonstrating essential version control workflows such as staging, committing, and pushing changes.

**Tech Stack:** HTML, CSS, JavaScript, Git, GitHub

## Features

* Real-time internet connectivity detection.
* Automatic UI updates when the connection status changes.
* Uses the `navigator.onLine` API and browser events.
* Simple, responsive, and user-friendly interface.
* Version controlled with Git and hosted on GitHub.
