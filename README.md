# Connectivity Check

A simple JavaScript-based web application that monitors and displays the user's internet connection status in real time. The application detects when the device goes online or offline and updates the UI accordingly, providing instant feedback about network connectivity.

## Features

* Real-time online/offline status detection
* Dynamic UI updates based on network state
* Lightweight and responsive design
* Built using vanilla JavaScript, HTML, and CSS
* User-friendly interface

## Technologies Used

* HTML
* CSS
* JavaScript
